🍔 QuickBite - Food Ordering Web Application

QuickBite is a full-stack food ordering web application that enables users to explore dishes, search food items, manage their cart, and place orders seamlessly using online payment or cash on delivery.

---

🚀 Features

- 🔍 Search and filter food items
- 🍽️ Browse menu by categories
- 🛒 Add to cart and manage items
- 💳 Secure online payment (Stripe integration)
- 💵 Cash on Delivery (COD) option
- 💬 AI-powered chatbot assistant (Gemini API)
- 📱 Fully responsive user interface

---

🛠️ Tech Stack

Frontend: React.js, CSS
Backend: Node.js, Express.js
Database: MongoDB
Integrations: Stripe, Gemini API

---

📂 Project Structure

QuickBite/
├── frontend/
├── backend/

---

⚙️ Environment Variables

Create a ".env" file inside the backend directory and add the following:

MONGO_URL=your_mongodb_connection_string
JWT_SECRET=your_secret_key
STRIPE_SECRET_KEY=your_stripe_secret_key
GEMINI_API_KEY=your_gemini_api_key

---

▶️ Run Locally

Backend

cd backend
npm install
npm run server

Frontend

cd frontend
npm install
npm run dev
